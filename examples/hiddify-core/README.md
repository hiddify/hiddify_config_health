# hiddify-core examples

Add examples here following the same structure as `../sing-box/` and `../xray/`.

Each subdirectory needs:
- `server.json` — server config template with `{{PLACEHOLDER}}` variables
- `client.json` — client config template
- `run.json`    — test metadata (`core`, `vars`, `checks`, etc.)

Set the `HIDDIFY_BIN` environment variable to the path of the hiddify-core binary.
